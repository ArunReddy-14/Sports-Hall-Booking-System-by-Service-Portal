# Sports-Hall-Booking-System-by-Service-Portal

A ServiceNow Service Portal application for booking sports halls online. The system allows users to book halls, check slot availability, calculate booking charges, track bookings, and request cancellations with refund processing.

## Features

- User login
- View available sports halls
- Book sports halls
- Dynamic booking charge calculation
- Slot availability check
- Automatic booking ID generation
- Multiple payment modes
- View booking history
- Cancel bookings
- Booking status tracking
- Refund status tracking
- Email notifications

## Technologies Used

- ServiceNow
- Service Portal
- HTML
- CSS
- AngularJS
- JavaScript
- GlideRecord
- Business Rules
- Notifications
